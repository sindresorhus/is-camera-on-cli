# is-camera-on-cli

> Check if the built-in Mac camera is on


## Install

```
$ npm install --global is-camera-on-cli
```

Requires macOS 10.11 or later.


## Usage

```
$ is-camera-on --help

  Example
    $ is-camera-on
    ✔ Camera is on

  Exits with code 0 if on and 1 if off
```
